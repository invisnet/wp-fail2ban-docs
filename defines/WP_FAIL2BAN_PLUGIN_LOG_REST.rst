WP_FAIL2BAN_PLUGIN_LOG_REST
---------------------------

.. versionadded:: 4.2.0

To enable logging plugin "REST" class messages, add the following to ``wp-config.php``:

.. code-block:: php

	define('WP_FAIL2BAN_PLUGIN_LOG_REST', true);
