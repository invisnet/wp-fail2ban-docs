WP_FAIL2BAN_PLUGIN_LOG_XMLRPC
-----------------------------

.. versionadded:: 4.2.0

To enable logging plugin "XMLRPC" class messages, add the following to ``wp-config.php``:

.. code-block:: php

	define('WP_FAIL2BAN_PLUGIN_LOG_XMLRPC', true);
