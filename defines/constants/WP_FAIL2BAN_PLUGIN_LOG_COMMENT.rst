WP_FAIL2BAN_PLUGIN_LOG_COMMENT
------------------------------

.. versionadded:: 4.2.0

To enable logging plugin "Comment" class messages, add the following to ``wp-config.php``:

.. code-block:: php

	define('WP_FAIL2BAN_PLUGIN_LOG_COMMENT', true);
