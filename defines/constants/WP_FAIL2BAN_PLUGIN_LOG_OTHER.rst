WP_FAIL2BAN_PLUGIN_LOG_OTHER
----------------------------

.. versionadded:: 4.2.0

To enable logging plugin "Other" class messages, add the following to ``wp-config.php``:

.. code-block:: php

	define('WP_FAIL2BAN_PLUGIN_LOG_OTHER', true);
